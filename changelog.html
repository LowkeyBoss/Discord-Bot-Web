Sonic Changelog

  • August 10, 2025
    Version 1.2.0 - Beta
    • Added new anonymous confession reply feature.
    • Improved moderation logging with more details.
    • Fixed bug where invite link was not generating correctly.
    • Performance improvements and minor UI tweaks.
  • July 22, 2025
    Version 1.1.0 - Beta
    • Introduced user and server stats commands.
    • Implemented blacklist feature for confessions system.
    • Added support for images in anonymous confessions.
  • June 30, 2025
    Version 1.0.0 - Initial Beta Release
    • Launched core moderation commands: kick, ban, timeout, purge.
    • Anonymous confession system with logging.
    • Bot invite and server join features.
