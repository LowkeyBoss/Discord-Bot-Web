Explore Sonic's Commands

Moderation

/kick
Remove a member from the server temporarily.
Usage: /kick @user [reason]
/ban
Ban a user from the server permanently.
Usage: /ban @user [reason]
/soft-ban
Softban a member (ban + immediate unban, deletes recent messages).
Usage: /soft-ban @user [reason]
/unban
Unban a user from the server.
Usage: /unban @user [reason]
/timeout
Timeout (mute) a member for a specified duration.
Usage: /timeout @user [duration]
/timeout-remove
Remove a timeout (mute) from a specified member.
Usage: /timeout-remove @user
/timeout-list
List all members currently timed out.
Usage: /timeout-list
/purge
Delete multiple messages at once.
Usage: /purge [number]
/mod-commands
Display all available moderation commands.
Usage: /mod-commands
/mod-config
Configure moderation system settings for the server.
Usage: /mod-config

Confessions System

/confess
Send an anonymous confession with optional image.
Usage: /confess [message] [attachment]
/confess-reply
Reply anonymously to a confession.
Usage: /confess-reply [confession ID] [message] [attachment]
/confess-blacklist
Blacklist a user from confessing or replying.
Usage: /confess-blacklist @user [reason]
/confess-unblacklist
Unblacklist a user from confessing or replying.
Usage: /confess-unblacklist @user [reason]
/remove-confession
Remove an anonymous confession.
Usage: /remove confession [number]
/confession-stats
Show statistics about the confession system in the server.
Usage: /confession-stats
/confession-leaderboard
Show the global leaderboard for confessions and replies across all servers.
Usage: /confession-leaderboard
/confession-config
Configure the confession system for the server.
Usage: /confession-config
/confession-info
View details of a specific confession.
Usage: /confession-info [number]

User & Server Info

/avatar
Displays a user's avatar.
Usage: /avatar @user
/user-info
Displays detailed information about a user.
Usage: /user-info @user
/server-info
Displays detailed information about the Discord server.
Usage: /server-info
/role-info
Displays detailed information about a specific role.
Usage: /role-info [role]
/joined
Displays when a user joined Discord and the server
Usage: /joined @user

Miscellaneous

/ping
Check the bot's latency and uptime.
Usage: /ping
/help
Show the help menu with all commands.
Usage: /help [command]
